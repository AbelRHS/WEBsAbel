01

Tipos de Listas

Una lista es un párrafo estructurado que contiene una serie de elementos. HTML define tres tipos de listas: ordenadas, no ordenadas y de definición.

<!-- Lista ordenada -->
<ol style="list-style-type: upper-roman">
  <li>Elemento I</li>
  <li>Elemento II</li>
</ol>

<!-- Lista no ordenada -->
<ul style="list-style-type: square">
  <li>Elemento</li>
</ul>

— Lista de Definiciones: Lenguajes

HTML
Siglas de HyperText Markup Language. Define una estructura básica y un código para el contenido de una página web como texto, imágenes, etc. Estándar a cargo de la W3C.
CSS
Cascading Style Sheets — lenguaje de hojas de estilo usado para describir la presentación de un documento escrito en HTML. Controla colores, tipografías, disposición y más.

03

Reproducción de Audio

Podemos encontrar en miles de webs audios que se reproducen al cargar la página, otros con botones de control, o en bucle. La compatibilidad varía según el navegador: Firefox no reproduce MP3 pero sí OGG.

autoplay

Al cargarse la página directamente empieza el audio.

controls

Podemos iniciar el audio y pararlo cuando queramos.

loop

Se inicia cuando le das play pero si no lo paras se reproduce en bucle.

muted

El audio por defecto aparece con la opción muted activada.

iframe — YouTube embed

Muy utilizado por YouTube para insertar reproductores externos.

<!-- Audio con controles -->
<audio controls loop>
  <source src="musica.mp3" type="audio/mpeg">
  <source src="musica.ogg" type="audio/ogg">
</audio>
04

Reproducción de Vídeo

Podemos encontrar en miles de webs vídeos que se reproducen al cargar la página, con controles, en bucle, o silenciados. También se puede insertar vídeo de YouTube con iframe.

controls

Podemos iniciar el vídeo y pararlo cuando queramos.

muted + loop

El vídeo aparece silenciado y se repite en bucle automáticamente.

<!-- Vídeo nativo -->
<video width="520" controls poster="preview.jpg">
  <source src="video.mp4" type="video/mp4">
</video>

<!-- YouTube con iframe -->
<iframe src="https://youtube.com/embed/ID"
  allowfullscreen></iframe>
05

Formularios Interactivos

Los formularios permiten recoger datos del usuario. Existen multitud de tipos de input: texto, email, número, contraseña, checkbox, radio y select, entre otros.

method="get"

Los datos se envían en la URL como parámetros. Visible en la barra de direcciones. Ideal para búsquedas.

method="post"

Los datos viajan en el cuerpo de la petición HTTP. No visibles en la URL. Ideal para datos sensibles como contraseñas.

autofocus

El campo con este atributo recibe el foco automáticamente al cargar la página, listo para escribir.

<form action="procesar.php"
      method="post">
  <input type="email"
         name="correo">
  <input type="submit"
         value="Enviar">
</form>

Etiquetas Semánticas

HTML5 introdujo etiquetas semánticas que dan significado a la estructura del documento. No solo organizan visualmente, sino que mejoran la accesibilidad y el SEO.

<header> <nav> <section> <article> <aside> <footer> <main> <figure> <mark>
